# Ghost AI

Next.js app scaffold for Ghost AI.

## Development

```bash
npm run dev
```

Open http://localhost:3000.

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` creates a production build.
- `npm run start` serves the production build.
- `npm run lint` runs ESLint.

## Project Structure

- `app/` contains App Router routes, layouts, and global styles.
- `public/` contains static assets served from the site root.
- `next.config.ts` contains Next.js configuration.

Before changing Next.js APIs or conventions, check the installed docs under
`node_modules/next/dist/docs/`.
